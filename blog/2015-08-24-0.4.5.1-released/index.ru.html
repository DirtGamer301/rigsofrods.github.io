Релиз 0.4.5.1!

Привет!

Команда разработчиков рада сообщить о выпуске первой порции исправлений для ветки 0.4.5 NextStable. Она содержит изменения, которые не попали в 0.4.5.0 и которые исправляют баги, обнаруженные уже после релиза NextStable.

0.4.5.1

Список изменений

  • Исправлено падение при пересечении ЖД-стрелок.

  • Добавлены тени для ещё большего числа моделей.

  • Убран лишний текст из окна оповещения.

  • Улучшено качество теней

  • Обновлены текстуры.

Скачать

Сборку для Windows можно скачать с сайта GitHub. Нынешние игроки RoR на Windows также могут использовать программу обновления, которая поставляется с 0.4.5.

Пользователи Linux могут использовать автоматические скрипты для загрузки и сборки последнего среза.

Спасибо за то что играешь в Rigs of Rods и надеемся, что ветка NextStable тебе понравится.
